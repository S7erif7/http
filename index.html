📖 What is an HTTP Request?

An HTTP Request is a message sent from the client (browser) to a web server asking for a resource.

طلب HTTP هو رسالة يرسلها المتصفح إلى خادم الويب لطلب صفحة أو ملف أو أي مورد آخر.

🌐 Request Structure

GET /download.html HTTP/1.1 Host: www.ethereal.com User-Agent: Mozilla/5.0 (Windows NT 5.1) Accept: text/html Connection: keep-alive

🔍 Explanation

Field English العربي
GET HTTP Method نوع الطلب
/download.html Requested Resource الملف المطلوب
HTTP/1.1 Protocol Version إصدار البروتوكول
Host Website Name اسم الموقع المطلوب
User-Agent Browser Information معلومات المتصفح
Accept Supported Content Types أنواع الملفات المقبولة
Connection Keep Connection Alive الإبقاء على الاتصال مفتوحًا

🚀 Common HTTP Methods

Method Description الوصف
GET Retrieve Data جلب البيانات
POST Send Data إرسال البيانات
PUT Update Resource تحديث مورد
DELETE Delete Resource حذف مورد

💡 Example from Wireshark

English
The browser requests the file /download.html from the server www.ethereal.com.

العربي
يقوم المتصفح بطلب الملف /download.html من الخادم www.ethereal.com.

📝 Quick Revision

  • ✅ HTTP is an Application Layer protocol.
  • ✅ HTTP uses TCP.
  • ✅ Default Port: 80.
  • ✅ HTTPS uses Port 443.
  • ✅ Every HTTP Request contains a Method.
  • ✅ The Host header specifies the destination website.
  • ✅ User-Agent identifies the client application.
