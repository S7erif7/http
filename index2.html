📥 HTTP Response

The HTTP Response is the message returned by the web server after receiving an HTTP Request.


استجابة HTTP هي الرسالة التي يرسلها خادم الويب إلى المتصفح بعد معالجة طلب HTTP.

🌍 HTTP Response Structure

HTTP/1.1 200 OK
Date: Thu, 13 May 2004 10:17:12 GMT
Server: Apache
Content-Length: 18070
Content-Type: text/html; charset=ISO-8859-1
Connection: Keep-Alive

<html>
...
</html>

🔍 Anatomy of an HTTP Response

Field English العربي
HTTP/1.1 Protocol Version إصدار البروتوكول
200 OK Status Code رمز حالة الطلب
Date Response Time وقت إرسال الاستجابة
Server Web Server Software برنامج الخادم
Content-Type Returned Content Type نوع المحتوى المُرسل
Content-Length Response Size حجم البيانات
Connection Connection State حالة الاتصال

📊 Common HTTP Status Codes

Code Meaning العربي
200 OK تم تنفيذ الطلب بنجاح
301 Moved Permanently إعادة توجيه دائمة
302 Found / Redirect إعادة توجيه مؤقتة
403 Forbidden الوصول مرفوض
404 Not Found الصفحة غير موجودة
500 Internal Server Error خطأ داخلي في الخادم

🧪 Example from Wireshark

HTTP/1.1 200 OK

200 OK means the server successfully processed the request.


200 OK تعني أن الخادم استقبل الطلب ونفذه بنجاح.

📦 Response Body

After the response headers, the server sends the requested content. This may include:

  • HTML Pages
  • CSS Files
  • JavaScript Files
  • Images
  • JSON Data
  • PDF Files

بعد الهيدرز يبدأ السيرفر في إرسال المحتوى الحقيقي المطلوب. قد يكون:

  • صفحات HTML
  • ملفات CSS
  • ملفات JavaScript
  • صور
  • JSON
  • ملفات PDF

📝 Quick Revision

✅ HTTP Response is sent by the server.
✅ Starts with a Status Code.
✅ Contains Response Headers.
✅ Contains the Response Body.
✅ Content-Length tells the browser how much data to expect.
✅ Content-Type tells the browser how to render the content.
